Over the past three days, significant progress was made in developing the GPS satellite visibility module. The backend now successfully computes satellite positions from RINEX navigation data, converts receiver coordinates to the ECEF reference frame, determines satellite azimuth and elevation angles, calculates receiver-to-satellite distances, and identifies visible satellites using a 10° elevation mask. The FastAPI backend has been tested successfully, returning complete satellite visibility information in JSON format. The next phase of development will focus on integrating these backend computations with the Electron-React desktop application, enabling users to input their receiver coordinates and visualize satellite visibility through an interactive graphical interface.
